# chrome-extension-template-v2
Chrome Extension Template for manifest v2

# 開発
## インストール
- TypeScript
```shell
npm i --save-dev typescript @types/jquery @types/chrome
```

- sass
```shell
npm i --save-dev sass
```

- gulp
```shell
npm i --save-dev gulp gulp-typescript gulp-sass fibers del
```

## ビルド
```shell
gulp build
```
